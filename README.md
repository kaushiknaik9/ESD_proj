# Edge AI Based Smart Farming System

This project implements an offline Edge-AI powered farming assistant using Raspberry Pi.

## Features
- Crop disease detection using CNN
- Pest detection and automation logic
- Crop yield prediction using ML
- Sensor data integration
- Streamlit web dashboard
- Fully offline (no internet required)

## Tech Stack
- Raspberry Pi
- Python
- TensorFlow Lite
- OpenCV
- Streamlit

## How to Run
```bash
source venv/bin/activate
streamlit run app/streamlit_app.py
